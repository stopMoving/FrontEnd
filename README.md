# 프론트엔드 레포

환영합니다!

## 초기세팅에 사용한 명령어

npm i styled-components
npm i react-router-dom
npm i axios
npx create-react-app

자세한 내용은 package.json 참고

## 초기 파일 구성

- 📷assets: 이미지, 폰트, 아이콘

- 🎡atom: Recoil 전역 상태 관리

- 🔌components: 재사용 컴포넌트들

- 🔖hooks: 커스텀 Hook 관리

- 📕lib: axios 인스턴스 같은거 두는 파일

- 📜pages: 사용자에게 비춰줄 페이지

- 🎈styles: CSS 스타일 모아두는 곳.
